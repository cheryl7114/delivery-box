# Frontend Source Structure

```
src/
├── assets/         # Images, logos, static files
├── components/     # Reusable UI components
├── pages/          # Page-level components (App, Home, etc.)
├── styles/         # CSS files
├── utils/          # Helper functions and utilities
├── hooks/          # Custom React hooks
├── services/       # API calls and external services
└── main.jsx        # Application entry point
```

## Folder Guidelines

- **assets/**: Store images, fonts, and other static resources
- **components/**: Reusable components like Button, Card, Navbar, etc.
- **pages/**: Top-level page components that represent routes
- **styles/**: All CSS/styling files
- **utils/**: Helper functions, constants, and utilities
- **hooks/**: Custom React hooks for shared logic
- **services/**: API integration and external service calls
